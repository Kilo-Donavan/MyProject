Les Balises DIV

La Balise DIV, est une balise neutre de type block. Elle me servira pour structurer ma page.

Les balises BLOCKS agissent sur un Axe Vertical et donc génère un retour à la ligne.
Je pourais toutes fois changer leurs comportements avec du CSS

Les Balises Inlineelles,agissent sur un Axe Horizontale et donc ne génère aucun retour a la ligne.


Je pourais toutes-fois changer leurs comportement avec du CSS.

Les Formulaires !

  • La balise permet de créer un formulaire
  • name : Permet de donner un nom au formulaire
  • action :Cette attribut est OBLIGATOIRE et permet de renseigner le script,la page vers laquelle l'utilisateur sera redirigé lors de l'envoi du formulaire. Cette page va reçevoir l'ensemble des données saisi par mon utilisateur
  • method :Permet de préciser quelle méthode [GET|POST] doit être utilisée par le navigateur pour transmettre les données saisies vers la page action.










Créer un lien en html

  • La balise "a"a permety de créer un lien vers une autre ressources. (images, vidéo, PDF, ZIP..)

  • href : Prend plusieurs valeurs : _blank pour ouvrir le lien dans une nouvelle fenêtre par exemple
  • title : Permet de donner un titre au lien.Ce titre sera afficher dans une petite bulle au survol du lien par la souris.
